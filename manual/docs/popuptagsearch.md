OPMATE Master에 연동된 노드에 부여된 전체 태그 정보를 조회할 수 있습니다.  
태그 키 Grid 에서 조회 할 태그 키를 선택하면, 태그 값 Grid 에 해당 태그 키로 부여된 태크 값이 모두 조회 됩니다.

![태그찾기][popup-tag-search]

### **조회 조건**

**선택 항목**

> **태그 키** : 조회할 태그가 부여된 시스템 구분과 태그 키를 확인하고 선택  


### **조회 결과**

**태그 값 Grid 영역**

> **시스템구분** : 태그를 부여한 시스템 구분. CSP/WorkPortal 등  
> **태그 키** : 선택한 태그 키 문자열  
> **태그 값** : 선택한 태그 키로 부여된 태그 값  

### **하단 버튼영역**
 
> <kbd class="btn-gray">&nbsp;ESC&nbsp;</kbd> : 팝업 닫기
