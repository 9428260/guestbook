## My Task

권한을 보유한 태스크 중 최대 6개 태스크의 최근 수행결과를 조회할 수 있습니다.   

![노드 목록][board]

### **제공 정보**

태스크의 정상 수행여부 및 수행 시간 등 요약 정보가 제공됩니다.

> **결과** : 태스크 수행 성공/실패/미수행 구분   
> **태스크 ID** : 태스크 ID  
> **결과 노드 수** : 실패, N/A, 성공 노드 수   
> **실행 유형** : 태스크가 실행된 방법 구분. 사용자/스케줄/태스크  
> **최근 수행일시** : 해당 태스크의 최종 실행시간  
> **결과보기** : 태스크 실행결과 상세 조회화면 링크  


### **Resources**

**관련 정보 조회 링크**

> **사용자 매뉴얼** : 콘솔 매뉴얼 페이지 링크  
> **Tag 모아보기** : 연동된 노드에 부여되어있는 전체 Tag 정보 일괄 조회  
> **홈페이지** : OPMATE 공식 홍페이지 링크
