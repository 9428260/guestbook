# LLM 기반 전문가 워크플로우를 활용한 실시간 P2P 에너지 거래용 다중 에이전트 강화학습

(LLM-Enhanced Multi-Agent Reinforcement Learning with Expert Workflow
for Real-Time P2P Energy Trading)

------------------------------------------------------------------------

## 초록 (Abstract 번역)

실시간 P2P(peer-to-peer) 전력 시장은 재생에너지 변동성과 수요 변화를
동적으로 반영하여 전력 가격을 즉각적으로 조정함으로써 경제적 이익을
극대화하고 전력망의 유연성을 향상시킨다.

그러나 대규모 개인화된 프로슈머 환경에서 전문가 지식을 확장 적용하는
데에는 다음과 같은 문제가 존재한다.

-   다양한 의사결정 요구
-   개인화된 모델링 프레임워크 부족

이를 해결하기 위해 본 논문에서는 **대규모 언어모델(LLM)과 다중 에이전트
강화학습(MARL)을 통합한 프레임워크(LLM‑MARL)** 를 제안한다.

주요 특징

-   LLM을 전문가(expert)로 활용
-   개인화된 전략 생성
-   CTDE(Centralized Training, Decentralized Execution) 패러다임 적용
-   모방학습(Imitation Learning) 기반 전략 지도

또한 대규모 P2P 네트워크 확장성 문제 해결을 위해 **차분
어텐션(differential attention) 기반 critic 네트워크**를 도입하였다.

실험 결과

-   LLM 전략이 인간 전문가를 효과적으로 대체
-   경제 비용 감소
-   전압 위반율 감소
-   안정성 향상

------------------------------------------------------------------------

## 1. 서론 (Introduction 번역)

P2P 에너지 거래의 등장으로 전력 사용자는 단순 소비자에서
**프로슈머(prosumer)** 로 변화하였다.

프로슈머는

-   전력을 생산하고
-   전력을 소비하는

두 역할을 동시에 수행한다.

하지만 다음 두 가지 문제가 존재한다.

### 1. 가상 레이어 문제

-   프로슈머는 반복적 전력 거래 능력이 부족
-   효율적인 에너지 관리 기술 부족

### 2. 물리적 전력망 문제

-   실제 배전망 안정성 보장 필요
-   전압 및 전력 흐름 제약 존재

또한 실제 전력 시장에서는 대부분 거래가 **Day‑Ahead Market**에서
이루어지지만

-   재생에너지 변동성
-   수요 불확실성

때문에 **Real‑Time Market**이 필요하다.

대표적인 전력 시장

-   Nord Pool
-   PJM
-   ERCOT
-   Australian NEM

## 기존 접근법의 한계

### Lyapunov Optimization

시간 결합 제약 완화 과정에서 근사 오차 발생

### Model Predictive Control (MPC)

확률적 환경에서 최적성이 떨어짐

### Model 기반 접근

정확한 시스템 모델 확보가 어려움

------------------------------------------------------------------------

## 강화학습(RL)의 등장

RL은 다음 장점이 있다.

-   불확실 환경 적응
-   전략 자동 생성

하지만 다음 문제가 있다.

-   trial‑and‑error 학습 필요
-   글로벌 시장 균형 수렴 어려움
-   개인화 정책 생성 어려움

------------------------------------------------------------------------

## 전문가 지식 기반 학습

이를 해결하기 위해

-   Optimization solver
-   Imitation learning

이 활용된다.

하지만

-   단일 전문가 의존
-   인간 전문가 비용 증가
-   실시간 대응 어려움

문제가 존재한다.

## LLM 활용

LLM은

-   추론 능력
-   의사결정 능력
-   일반화 능력

을 제공하며

**에너지 전문가 역할을 대체할 수 있다.**

## 논문의 핵심 기여

1.  **LLM‑MARL 통합 프레임워크**
2.  **프로슈머 맞춤 LLM 전문가 워크플로우**
3.  **Wasserstein 기반 모방 MARL 알고리즘**
4.  **Differential Multi‑Head Attention Critic**

## LLM Expert Workflow

LLM 기반 전문가 시스템은 다음 단계로 동작한다.

### 1. Model Generation Expert

프로슈머 장치와 운영 조건 이해

### 2. Code Generation Expert

최적화 모델 코드 생성

### 3. Iterative Correction Expert

코드 오류 자동 수정

### 4. Energy Trading Integration Expert

P2P 거래 변수 통합

------------------------------------------------------------------------

## LLM 전문가 활용 예시

예: 한 가정 프로슈머

조건

PV 발전 = 4kW\
부하 = 2kW\
배터리 SOC = 70%\
P2P 가격 = 110원\
Grid 판매가격 = 80원

잉여전력 = 2kW

LLM 전문가 판단

1.  P2P 가격이 Grid 가격보다 높음
2.  배터리는 충분히 충전됨
3.  따라서 P2P 판매가 최적

전략

Sell 2kW to P2P market

이 전략은 MARL 학습에서 **expert action**으로 사용된다.

------------------------------------------------------------------------

## MARL 학습 방식

에이전트 행동과 전문가 행동 차이를

**Wasserstein distance**로 계산하여

정책을 업데이트한다.

초기 학습

Agent ≈ Expert

후기 학습

Agent self exploration

------------------------------------------------------------------------

## SEAPAC 시스템 적용 예

LLM은 다음 Agent 전략을 생성할 수 있다.

SmartSeller‑Agent - 잉여 태양광 전력 판매 전략 생성

StorageMaster‑Agent - ESS 충방전 전략 생성

EcoSaver‑Agent - 사용자 절약 행동 추천

즉

**LLM = 에너지 전문가 + 전략 생성 엔진**
